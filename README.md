# 👥 Facebook UI Clone

Uma réplica responsiva e detalhada da interface principal do Facebook, desenvolvida com **HTML5** e **CSS3** para consolidar o aprendizado em componentização e layouts de alta complexidade.

> 🚀 Este repositório faz parte do meu portfólio de estudos práticos em desenvolvimento front-end estático.

---

## ✨ Demonstração

O projeto foi publicado e pode ser visualizado online através do link abaixo:
🔗 [Clique aqui para acessar o projeto online](https://github.com/Francacommit/facebook-interface-replica)

---

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica para os blocos de conteúdo e o feed de notícias.
*   **CSS3:** Estilização avançada com Flexbox (alinhamento de 3 colunas) e efeitos modernos de hover.
*   **Google Fonts:** Utilização da fonte *Poppins* de forma otimizada para garantir a melhor performance.

---

## 📂 Organização do Projeto

Seguindo a mesma arquitetura modular dos meus projetos anteriores, os arquivos de estilo foram divididos para facilitar a manutenção:
```text
├── style/
│   ├── index.css  # Variáveis globais, barra de navegação superior e feed
│   └── media.css  # Regras de responsividade (@media queries) para telas menores
├── index.html     # Estrutura principal da página
└── README.md      # Documentação do repositório
```

---

## 📐 Desafios Técnicos Superados

*   **Layout Tripartido (3 Colunas):** Implementação bem-sucedida do layout clássico de desktop com menu lateral, feed central e painel direito de contatos usando Flexbox.
*   **Responsividade Dinâmica:** Ocultação inteligente das barras laterais em telas menores (`media.css`) para garantir que o feed de notícias fique centralizado e confortável em dispositivos móveis.
*   **Padronização de Commits:** Utilização do padrão *Conventional Commits* (ex: `feat(layout): ...`) para manter o histórico de desenvolvimento organizado e alinhado ao mercado.
